Union Find

Implementation Of Quick Find

  • Implementation
    • Union(p, q): change all entries whose id equals id[p] to id[q]
    • Connected(p, q): judge if id[p]==id[q]
    • find(p):return p
  • Too Slow !

Quick Union

  • Balance By linking root of smaller tree to root of larger tree
  • lazy union, only union the root of two trees

Weighted Qucik Union With Path Compression

  • Quick union too slow when the tree grow too deep
  • keep track of each tree, use another array to store the depth of each tree
  • Compress path when finding the root of a node

Summary

algorithm initialize union find
quick-find N N 1
qiick-union N N+ N
weightedQuickUnion N lg N lg N
  • In fact, the speed of weightedQucikUnion is almost linear in actual usage!
